# POO — Programmation Orientée Objet en Python  

Ce dépôt regroupe les exercices, tests et notes réalisés dans le cadre du brief hebdomadaire consacré à la Programmation Orientée Objet (POO) en Python.  
Ce travail s’inscrit dans mon parcours de formation Data Analyst.

---

## 📁 Structure du projet

POO/

---

## 🎯 Objectifs 

### 🔸 Comprendre les bases de la POO
- Définition de classes et création d’objets  
- Attributs d’instance et de classe  
- Méthodes, encapsulation, représentation (`__str__`, `__repr__`)  

### 🔸 Approfondir les mécanismes avancés
- Héritage simple et multiple  
- Polymorphisme et redéfinition de méthodes  
- Comparaison d’objets (`__lt__`, `__eq__`)  
- Tri personnalisé et cohérence des structures  

### 🔸 Développer des compétences utiles au métier de Data Analyst
- Organisation propre du code  
- Tests unitaires pour fiabiliser les traitements  
- Utilisation de Git/GitHub pour versionner et documenter un projet  
- Bonne pratique de structuration de projet Python
